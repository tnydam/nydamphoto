## Deployment to GitHub Pages
1. Create repo: `nydamphoto` (or use existing)
2. Upload all files to root of `main` branch
3. Go to Settings → Pages → Source: main branch / root
4. Custom domain: nydamphoto.com
5. Add CNAME file with: nydamphoto.com
